# 📊 Sentix Insights Dashboard


## 📊 Project Overview
This project provides a comprehensive sentiment analysis across major social media platforms (Facebook, Instagram, and Twitter). The dashboard transforms raw social engagement data into actionable insights, focusing on audience mood trends, platform-specific activity, and peak engagement timing.

The goal is to help brand managers understand public perception, optimize posting schedules based on heatmaps, and identify which platforms drive the most positive community interaction.

## 🛠️ Tools Used
- Excel (Data Cleaning & Pivot Tables)
- Power BI / Tableau (Data Visualization & Dashboarding)

  
## 💡 Data Insights
- Neutral Dominance: A significant portion of the conversation remains neutral, though positive sentiment shows consistent growth throughout the mid-year (June–September).
- Platform Engagement: Twitter leads in total engagement volume, while Instagram maintains the most consistent daily posting activity.
- Peak Timing: Heatmap analysis reveals that Evening and Night slots across all platforms see the highest activity levels, suggesting that's when the audience is most reactive.
- Total Reach: With over 46K total engagements from only 711 posts, the data suggests a high engagement-per-post ratio.
